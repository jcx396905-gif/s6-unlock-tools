# S6 Software Installation Restriction Removal Tool

Tool for removing software installation restrictions on Samsung S6 devices.

## Contents

- `S6解除软件安装限制工具_V3.7.exe` - Main executable
- `patch/` - Patch files including:
  - `EEBBK-11.zip` - System patch
  - `Magisk-v25.2.apk` - Magisk root app
- Documentation (Chinese):
  - `关于系统低于官网无法下载的解决方法(20221116).docx`
  - `论坛.docx`

## Warning

Use at your own risk. Modifying system restrictions may void warranty.
